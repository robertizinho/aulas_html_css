ID e CLASS

ID

o ID funciona como um cpf.Assim como só posso ter um numero de cpf, cada elementosó pode ter um valor de ID. O cpf é unico, ou seja niguem pode ter um cpf igual ao meu.Isso significa que o ID também é unico,então nao e possivel repetir o mesmo valor duas vezes

CLASS

O CLASS funciona de maneira oposta ao ID.Diferente do id em que voce pode ter um valor,no class voce pode ter varios valores.tambem diferentes do id,em que não se pode repetir um valor,no class voce pode utilizar o mesmo valor para varios elementos
